  <action>
Replace OvercookedLocalView (lines 793-851) with the new instance-method API. The class keeps its `@register_feature_type("local_view", scope="overcooked")` decorator and docstring.

**Changes:**

1. Add `n_extra_channels = 8` as a class attribute (replaces `extra_n_channels` classmethod).

2. Add `__init__(self, scope, env_config=None)` that:
   - Calls `super().__init__(scope, env_config)` (this sets self.scope, self.env_config, etc.)
   - Imports `from cogrid.core.grid_object import object_to_idx`
   - Sets `self._onion_id = object_to_idx("onion", scope=scope)`
   - Sets `self._tomato_id = object_to_idx("tomato", scope=scope)`
   - Sets `self._capacity = 3`
   - Sets `self._cook_time = xp.float32(20.0)`

3. Add `extra_channels(self, state, H, W)` instance method that:
   - Extracts `pot_pos = state.pot_positions`, `pot_contents = state.pot_contents`, `pot_timer = state.pot_timer`
   - Computes `pot_rows = pot_pos[:, 0]`, `pot_cols = pot_pos[:, 1]`
   - Computes the 8 per-pot value arrays (same math as the old closure):
     - `n_items = xp.sum(pot_contents > 0, axis=1)`
     - `is_empty = (n_items == 0).astype(xp.float32)`
     - `is_partial = ((n_items > 0) & (n_items < self._capacity)).astype(xp.float32)`
     - `is_cooking = ((n_items == self._capacity) & (pot_timer > 0)).astype(xp.float32)`
     - `is_ready = ((n_items == self._capacity) & (pot_timer == 0)).astype(xp.float32)`
     - `fill_norm = n_items.astype(xp.float32) / self._capacity`
     - `timer_norm = pot_timer.astype(xp.float32) / self._cook_time`
     - `n_onion = xp.sum(pot_contents == self._onion_id, axis=1).astype(xp.float32)`
     - `n_tomato = xp.sum(pot_contents == self._tomato_id, axis=1).astype(xp.float32)`
   - Uses `self._scatter_to_grid(H, W, pot_rows, pot_cols, values)` for each of the 8 value arrays to produce 8 `(H, W)` grids
   - Stacks them: `return xp.stack(layers, axis=-1)` to produce `(H, W, 8)` float32 array

4. Remove the old `extra_n_channels` classmethod entirely.
5. Remove the old `build_extra_channel_fn` classmethod entirely.

**Key difference from old code:** The old code returned `list[(H,W)]`. The new code returns `ndarray(H, W, 8)` via `xp.stack(layers, axis=-1)`. The bridge in `build_feature_fn` will unstack it.

The `_N_POT_CHANNELS = 8` module constant should remain (used elsewhere for documentation).
